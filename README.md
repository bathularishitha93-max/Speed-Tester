# ⌨️ Python Typing Speed Tester

## 📌 Project Description

Python Typing Speed Tester is a simple command-line application that helps users measure their typing speed and accuracy. It randomly displays a paragraph, records the time taken to type it, and calculates:

- Typing Speed (Words Per Minute - WPM)
- Typing Accuracy
- Performance Rating

This project is suitable for beginners learning Python and demonstrates the use of loops, functions, lists, string operations, random module, and time module.

---

## 🚀 Features

- Random paragraph generation
- Measures typing time
- Calculates Words Per Minute (WPM)
- Calculates typing accuracy
- Gives performance feedback
- Allows multiple test attempts
- Beginner-friendly code

---

## 🛠 Technologies Used

- Python 3.x
- time module
- random module

No external libraries are required.

## ▶️ How to Run

1. Install Python 3.
2. Save the program as:

```
typing_speed_tester.py
```

3. Open Terminal or Command Prompt.

4. Navigate to the project folder.

5. Run:

```bash
python typing_speed_tester.py
```
